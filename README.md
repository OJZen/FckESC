## 天邑登录

简洁、纯净的第三方天翼校园网登录程序。

## 运行环境

系统：Windows 7 / 8 / 8.1 / 10 / 11

运行框架：[.NetFramework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/thank-you/net48-chs)

## 开发环境

IDE： Visual Studio 2022

平台：WPF

系统：Windows 10

以上仅为我个人的开发环境，理论上只要能跑VS2022就行。
